## An optimized, simpler fork of Doido Engine, geared towards higher customization on a mod to mod basis!

### VERSIONS NECESSARY
```
haxe 4.3.3
lime 8.1.2
openfl 9.3.3
flixel 5.8.0
flixel-addons 3.2.2
flixel-ui 2.6.1
hxdiscord_rpc https://github.com/MAJigsaw77/hxdiscord_rpc
```
